# 다신샵 프론트엔드 (클론)

[다신샵 상품 페이지](https://dshop.dietshin.com/goods/view.asp?JP=GDN1_C&g=14211)를 참고한 React + TypeScript 클론 프로젝트입니다.

## 기술 스택

- **React 18** + **TypeScript**
- **Vite** (빌드 도구)

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 으로 접속합니다.

## 빌드

```bash
npm run build
```

## 구현된 화면

- **헤더**: 로고(DASHIN), 로그인/회원가입, 장바구니, 회원가입 혜택 안내
- **네비게이션**: 전체 카테고리, 찐살복구 특가, BEST, SALE, NEW 등
- **상품 상세**: [15+15 특가] 닭신 오븐구이 닭안심살 7종 골라담기
  - 가격 (59,400원 / 51% 할인)
  - 필수옵션1·2 (7종 맛 선택)
  - 배송 정보 (새벽배송 / 택배배송)
  - 장바구니 담기 / 구매하기
- **탭**: 상세설명, 구매정보, 상품후기(17019), 제품정보
- **상품후기** 샘플 목록

백엔드(dasin-backend)와 연동하려면 API 베이스 URL을 환경 변수 등으로 설정해 사용하면 됩니다.
